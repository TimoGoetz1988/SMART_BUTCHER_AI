# SMART_BUTCHER_AI – Metzgerei-Produktionsvorhersage

Dieses Projekt zeigt, wie eine Metzgerei mit Hilfe von KI ihre tägliche Produktion besser planen kann.

Auf Basis historischer Verkaufszahlen werden Vorhersagen für die nächsten Tage erstellt. Ziel ist es,
Überproduktion und Warenabschreibung zu reduzieren und gleichzeitig Engpässe zu vermeiden.

> Projekt im Rahmen der Weiterbildung **KI-Manager (DEKRA)** von Timo Götz.

---

## Inhalt des Repositories

- `smart_butcher_forecast.ipynb`  
  Jupyter Notebook mit der vollständigen Analyse und dem Vorhersagemodell
  (Datenaufbereitung, Modellierung, Visualisierung der Prognosen).

(Optional)
- `requirements.txt`  
  Liste der Python-Pakete, die für das Notebook benötigt werden.
